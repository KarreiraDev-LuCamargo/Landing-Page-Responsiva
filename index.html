Desenvolvimento Front-End

Este setor é responsável pela parte visível de um site ou aplicação. Os profissionais desta área utilizam tecnologias como HTML, CSS e JavaScript para criar a interface, o layout e a experiência de interação do utilizador.

Desenvolvimento Back-End:

Foca-se no "bastidor" de uma aplicação, ou seja, tudo o que não é visto pelo utilizador. Esta área envolve a gestão de servidores, bancos de dados e a lógica de programação que conecta todos os componentes

Desenvolvimento Full-Stack:

Um profissional full-stack é versátil e capaz de trabalhar tanto no front-end quanto no back-end, combinando conhecimentos das duas áreas.

Designer UX/UI (User Experience/User Interface):

Este profissional desenha a aparência e a usabilidade do site, assegurando uma boa experiência para o utilizador. Eles criam wireframes, protótipos e designs visuais.

Gestor de Projetos:

Responsável por testar o site ou aplicação, garantindo que todas as funcionalidades estão a operar corretamente e sem erros (bugs).
